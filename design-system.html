Cores

07 ── Princípios & Voz

Princípios de Design

01
Noir sem Generic
Preto-arroxeado + ouro + crimson + pergaminho. Nunca cinza neutro ou branco puro. A escuridão é atmosfera, não falta de cor.
02
Dossiê como Metáfora
Todo card de conteúdo é um documento de caso. Topo afiado + base arredondada + carimbo "ARQUIVO CONFIDENCIAL". O raio assimétrico é inviolável.
03
Ouro = Ação Principal
Gold aponta para o que o jogador deve fazer agora. Crimson aponta para perigo. Nunca usar gold como decoração genérica — ele tem peso semântico.
04
Tipografia Teatral
Playfair Display em itálico é a assinatura dramática do jogo. Courier Prime é o arquivo. Nunca trocar por fontes system ou sans-serif genérico.
05
Motion com Intenção
Spring para feedback tátil imediato. Dramático para revelações. Snappy para navegação. Nunca animar por animar — cada easing tem um papel.
06
Identidade dos Suspeitos
As 6 cores dos suspeitos são canônicas. Nunca alterar, nunca usar fora de contexto. São a linguagem visual de identidade do jogo.

Do & Don't

✓ Fazer
✓Usar --radius-dossier em todos os cards de conteúdo.
✓Playfair Display em itálico para citações e ênfase narrativa.
✓Aplicar gold apenas em elementos de ação ou destaque semântico.
✓Usar --gold-rgb (o canônico) para construir glows com rgba().
✓Usar a cor do suspeito via --suspect-N (não hardcode).
✗ Não Fazer
✗Hardcode de cores fora dos tokens — sempre usar variáveis CSS.
✗Usar Inter, Roboto, Arial ou fontes system no lugar das famílias tipográficas do sistema.
✗Usar gold como cor decorativa sem função semântica.
✗Aplicar border-radius genérico (8px uniforme) em cards — o dossiê tem raio assimétrico.
✗Criar novos valores de ouro fora da escala gold-700→300.

Inventário de Ícones (Emoji)

O sistema de ícones é 100% emoji. Nenhum SVG customizado ou icon font necessário.
